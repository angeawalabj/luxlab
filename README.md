# LuxLab 🔬

> Open-source light physics simulation platform

Simulate light in all its states — geometric, wave, quantum, nuclear, spectroscopic —
in a single unified environment. Built for students, researchers, and professionals.
Designed to run offline on low-end hardware.

## Modules
- Geometric Optics
- Wave Optics  
- Quantum Optics
- Nuclear / Radiation
- Spectroscopy
- Electromagnetism

## Stack
- Frontend: React + Vite + Three.js + TailwindCSS
- Physics engine: Rust → WASM
- Collaboration: Yjs + WebRTC (works LAN, no internet needed)
- File format: .lux (open JSON-based format)

## Getting Started
```bash
npm install
npm run dev
```

## Vision
One tool. All of light. For everyone — from Cotonou to MIT.
